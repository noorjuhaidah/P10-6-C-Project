# P10-6-C-Project
Class Management System (CMS)

Project Overview

The Class Management System (CMS) is a simple database management system designed to manage student records for a class. It allows users to perform Create, Read, Update, and Delete (CRUD) operations on student records stored in a text file. This system facilitates efficient management of student data for educational purposes.

The CMS interacts with a text-based database file, where each studentâ€™s record includes their unique ID, name, programme, and marks. The program supports features such as sorting, querying, updating, deleting, and saving records.

Main Features
    - Insert: 
        Add new student records to the database.
    - Show All: 
        Display all student records stored in the database.
    - Query: 
        Search for a record based on the student ID.
    - Update: 
        Modify student information such as marks, name, or program.
    - Delete: 
        Remove a record based on the student ID.
    - Sort: 
        Sort records by student ID or mark.
    - Save: 
        Persist changes to the file database.
    - Summary: 
        Display total number of students, average mark, highest and lowest mark with student details.
